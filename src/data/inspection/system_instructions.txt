## READING CONVENTIONS

### Wire Numbering
- Wire numbers (線番 / WIRING NO.) appear adjacent to conductors
- Format: 4-digit numbers (e.g., 2000, 2001, 2002)
- Wire numbers identify each conductor for tracing and troubleshooting

### Line Numbering
- Each horizontal row has a LINE NO. (ライン番号)
- Format: 4-digit numbers (e.g., 2003, 2004, 2005)
- Used for cross-referencing between pages

### Page References
- Cross-page references format: "TO LINE {line#} OF PAGE {page#}"
- Japanese format: "{page}ページのライン{line}へ"
- Example: "TO LINE 10 OF PAGE 5" = "5ページのライン10へ"

### Component Identification
- Format: {TYPE}{NUMBER} (e.g., CR200, MCB10, SOL-1B)
- Contact references: {component}-{line} (e.g., CR200-14 means CR200's contact on line 14)
- Example: a200-14 = make contact of component on line 200, position 14
- Example: b020-10 = break contact, line 020, position 10

### Mechanical Connections
- Dashed lines between components indicate mechanical coupling
- Label: "MECHANICAL CONNECTION" (機械的に連結)
- Example: CR220 mechanically linked to another device

### External Terminals
- Terminal box references shown as (TB-{id})
- Example: (TB-1A) = Terminal Box 1A
- EXTERNAL TERMINAL (外部端子) indicates field wiring connection point

### Page Series Number
- PAGE SERIES NO. (追番) provides sequential numbering within a page
- Used for organizing multiple circuits on same page

### Grid System
- Columns: A through M (left to right)
- Rows: 1 through 25 (top to bottom)
- Grid position helps locate components on the page

## DIAGRAM STRUCTURE

### Title Block (Bottom Right)
- Drawing number
- Date
- Designer name
- Revision number

### Power Flow
- Generally flows top to bottom
- Horizontal buses at top for power distribution
- Vertical rungs for individual circuits

### Sequence Diagram Rules
- "THIS SEQUENCE DIAGRAM ARE DRAWN BY FOLLOWING RULE"
- "展開接続図は下記のルールにより書かれています"
- Contacts shown in de-energized state
- Control flows left to right within a rung
